Punk Documentation

Base Types


Number:

Any Real number. This includes the natural numbers such a 10 or fractional amounts like 5.777. Internally, we store these numbers as longs and doubles respectfully. Computationally, the numbers are 8 bytes (64 bits)

As punk isn't a strongly typed language, the only thing you need to do to set a variable to a long is set it equal. The parser will determine it's type by the input

x = 5.6

y = 5

Doubles have range

$ \approx \pm{{'{'}}5.0 \times 10^{{'{'}}324{{'}'}} \pm 1.7 \times 10^{{'{'}}308{{'}'}} $

Longs have a range of

$ {{'{'}} -2,147,483,648 to 2,147,483,647{{'}'}} $

String:

These are elements that are meant to represent text.Any type declared as a string must have double quotes(" ") surrounding the text content.

Data:

A data type is an array of numbers. No other types are supported as an element of data

You can declare an array in 2 ways:

x = [45,43.0,7.11,7]

or x = [-100...100]

The latter will initialize an array with 200 elements: [-100, -99....99,100] .

Sequence:

A sequence operates on Data. You can look at a sequence as a Action that is done on each and every element in the data.

Please note that applying a sequence transforms the value in any data variable
